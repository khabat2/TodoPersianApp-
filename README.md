# اپ اندرویدی «کارهای من»

یک اپ بومی اندروید به زبان فارسی برای مدیریت کارهای روزمره با رابط راست‌به‌چپ، تاریخ شمسی، یادآور آلارمی، و امکان افزودن تصویر و صدای اختصاصی برای هر کار.

## امکانات

- رابط کاربری فارسی و راست‌به‌چپ
- طراحی شیک با هدر گرادیانی و کارت‌های مدرن
- افزودن، ویرایش، حذف و تکمیل کارها
- ثبت توضیحات برای هر کار
- انتخاب تاریخ و ساعت بر اساس تقویم شمسی
- زمان‌بندی یادآور با AlarmManager
- اعلان و آلارم هنگام رسیدن زمان کار
- امکان انتخاب صدای اختصاصی برای آلارم هر کار
- امکان افزودن تصویر به هر کار
- ذخیره‌سازی محلی با SharedPreferences، بدون نیاز به اینترنت یا سرور

## اجرای پروژه

1. پوشه `TodoPersianApp` را در Android Studio باز کنید.
2. اجازه دهید Gradle Sync انجام شود.
3. اپ را روی شبیه‌ساز یا گوشی اندرویدی اجرا کنید.
4. برای اندروید ۱۳ به بالا، اجازه Notification را بدهید.
5. در بعضی گوشی‌ها برای آلارم دقیق باید از تنظیمات سیستم، اجازه exact alarm یا اجرای پس‌زمینه را فعال کنید.

## نکته‌های فنی

- زبان پروژه: Java
- حداقل نسخه اندروید: Android 6.0 / API 23
- نسخه هدف: API 35
- وابستگی خارجی ندارد؛ همه UI به‌صورت native و programmatic پیاده‌سازی شده است.
- تبدیل تاریخ میلادی/شمسی در کلاس `PersianDate.java` انجام می‌شود.

## فایل‌های اصلی

- `MainActivity.java`: رابط کاربری، افزودن/ویرایش کارها، انتخاب تصویر و صدا
- `Task.java`: مدل داده کار
- `TaskStore.java`: ذخیره و بازیابی محلی
- `PersianDate.java`: تبدیل و نمایش تاریخ شمسی با اعداد فارسی
- `AlarmScheduler.java`: زمان‌بندی یادآورها
- `AlarmReceiver.java`: دریافت آلارم
- `AlarmSoundService.java`: نمایش اعلان و پخش صدای آلارم
